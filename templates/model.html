LOGISTIC REGRESSION

  • Logistic Regression measures the relationship between the dependent variable (label to be predicted) and the one or more independent variables (input features), by estimating probabilities using its underlying logistic function.
  • These probabilities must be converted into binary values to actually make a prediction
  • This is the task of the logistic function, also called the sigmoid function.The Sigmoid-Function is an S-shaped curve that can take any real-valued number and map it into a value between the range of 0 and 1, but will not exceed those limits
  • These values between 0 and 1 will be converted into either 0 or 1 using a threshold classifier


    • K NEAREST NEIGHBOURS

    • All instances will correspond to points in an n-dimensional feature space
    • Each of the instances is represented with a set of numerical attributes.
    • Each of the training data has a set of vectors and a class label which is associated with each of the vectors
    • Classification is performed by comparing feature vectors of different K nearest points
    • Select the K-nearest neighbours to E
    • Allocate E to the most common class among its K-nearest neighbors
